## Step 1: Download and Login to GitHub Desktop

Download **GitHub Desktop** from the official website and sign in with your GitHub account.

<p align="center">
  <img width="900" alt="Download GitHub Desktop" src="images/tutorial/codestructure/download-github.png" />
</p>

---

## Step 2: Clone Your Repository

Cloning creates a local copy of your remote repository so you can edit files directly on your computer.

**2.1.** In GitHub Desktop, choose **Clone a repository from the Internet**.

<p align="center">
  <img width="900" alt="Clone repository option" src="images/tutorial/codestructure/clone-repo.png" />
</p>

**2.2.** Select the repository you want to clone, choose a local path, then click **Clone**.

<p align="center">
  <img width="900" alt="Clone confirmation dialog" src="images/tutorial/codestructure/clone.png" />
</p>

---

## Step 3: Organize Your Code

### Fetch and Pull

**Fetch** — Check for new changes from the remote repository (GitHub) **without** modifying your local files.

<p align="center">
  <img width="900" alt="Fetch from remote" src="images/tutorial/codestructure/fetch-repo.png" />
</p>

**Pull** — Download the changes from the remote repository and **merge** them into your current branch.

<p align="center">
  <img width="900" alt="Pull changes" src="images/tutorial/codestructure/pull-repo.png" />
</p>

---

### Create and Move Files

**3.1.** Open the repository folder in File Explorer using the **Show in Explorer** option.

<p align="center">
  <img width="900" alt="Show repository in Explorer" src="images/tutorial/codestructure/show-in-explorer.png" />
</p>

**3.2.** Create a new folder to organize your code by topic or category.

<p align="center">
  <img width="900" alt="Create a new folder" src="images/tutorial/codestructure/create-new-folder.png" />
</p>

**3.3.** Cut the code files you want to move.

<p align="center">
  <img width="900" alt="Cut code files" src="images/tutorial/codestructure/cut.png" />
</p>

**3.4.** Paste them into the new folder.

<p align="center">
  <img width="900" alt="Paste files into new folder" src="images/tutorial/codestructure/paste.png" />
</p>

---

### Commit and Push

**3.5.** Back in GitHub Desktop, enter a **Commit Message** describing your changes, then click **Commit to main**.

> **Note:** A commit message is required. For example: `"update leetcode"`.

<p align="center">
  <img width="900" alt="Enter commit message" src="images/tutorial/codestructure/commit-message.png" />
</p>

**3.6.** Click **Push origin** to send your local commits to the remote repository on GitHub.

<p align="center">
  <img width="900" alt="Push commits to GitHub" src="images/tutorial/codestructure/push.png" />
</p>

---

### Verify on GitHub

**3.7.** Press **F5** to refresh your GitHub page. The new folder should now appear in your repository.

<p align="center">
  <img width="900" alt="New folder visible on GitHub" src="images/tutorial/codestructure/new-folder.png" />
</p>

<p align="center">
  <img width="900" alt="Final repository structure" src="images/tutorial/codestructure/result.png" />
</p>
